# Through the Eyes of Claude Monet

> 一个致力于探索克劳德·莫奈生平与艺术作品的网站，使用 GitHub Pages 搭建。

## 🌟 项目概述

本网站展示了法国画家 Claude Monet 的生平与艺术作品 ——  
- **Biography** 页面介绍莫奈的一生与重要历程。  
- **Artworks / Map** 页面提供莫奈作品与其地理／历史背景。  
- **About** 页面说明网站的创建目的与设计思路。  

总体来说，本项目是一个关于莫奈艺术与人生的线上导览，适合艺术爱好者与研究者浏览。

## 📦 使用方式 / 浏览方式

1. 可直接访问 / 部署到你自己的 GitHub Pages：  
   > https://你的用户名.github.io/Arts-Website_Claude-Monet/  
2. 若需在本地运行／查看，可通过静态服务器（例如 VS Code Live Server）打开根目录的 `index.html`。  

## 🛠 开发与部署

- 本项目基于静态 **HTML + CSS +（如有）JavaScript**。  
- 使用 GitHub Pages 部署 —— 无需额外后端服务。  
- 若要修改网站内容，只需编辑对应的 HTML／Markdown 文件 → commit → push → GitHub Pages 将自动重新部署。
